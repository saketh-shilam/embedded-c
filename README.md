# Embedded C Projects 🚀

##  About This Repository

This repository contains my Embedded C programs and projects developed during my learning journey in microcontrollers and embedded systems.

The focus is on practical implementation of concepts like GPIO control, delay generation, and hardware interfacing using the 8051 microcontroller.

---

##  Objectives

* Learn Embedded C programming
* Understand microcontroller basics (8051)
* Practice hardware interfacing
* Build strong fundamentals for VLSI & Embedded Systems

---

##  Projects Included

###  LED Blinking Pattern (8051)

* Alternating LED pattern using Port 1
* Uses delay function for timing control

**Working:**

* Sends `0x55` → alternate LEDs ON
* Sends `0xAA` → reverse pattern
* Repeats continuously

1. Open project in Keil uVision
2. Compile the code
3. Upload HEX file to microcontroller
4. Observe LED blinking pattern
